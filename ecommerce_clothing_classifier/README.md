# `ecommerce_clothing_classifier`

Small computer vision exercise built with PyTorch on FashionMNIST.

## Project

- `ecommerce_clothing_classifier.py`
  Defines a compact convolutional neural network, trains it for one epoch, and reports accuracy, precision, and recall on the test set.

## Libraries used

- `torch`
- `torchvision`
- `torchmetrics`
- `numpy`

## What I practiced here

- CNN fundamentals
- DataLoader-based training loops
- multiclass evaluation with per-class metrics

## Running notes

- The script downloads FashionMNIST automatically on first run.
